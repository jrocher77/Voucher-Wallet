## 🧪 Comment Tester Vos Couleurs

### Dans l'app
1. **Créer un nouveau bon**
2. **Aller à "Couleur de la carte"**
3. **Choisir les couleurs**
4. **Observer l'aperçu**
   - ✅ Si lisible → Bon contraste
   - ⚠️ Si avertissement → Contraste faible
   - ❌ Si invisible → Contraste nul

### Avec TextColorExampleView
```swift
// Dans votre code ou preview
TextColorExampleView()
```
→ Voir 8 exemples avec calculs de contraste en direct !

### Formule manuelle
```swift
// 1. Calculer la luminosité
L1 = luminance(couleur1)
L2 = luminance(couleur2)

// 2. Calculer le ratio
ratio = (max(L1, L2) + 0.05) / (min(L1, L2) + 0.05)

// 3. Vérifier
if ratio >= 7.0 { print("✅ AAA - Excellent") }
else if ratio >= 4.5 { print("✅ AA - Bon") }
else if ratio >= 3.0 { print("⚠️ A - Acceptable") }
else { print("❌ Insuffisant") }
```

## 💡 Conseils de Design

### Règles d'Or

1. **Contraste élevé = Meilleure lisibilité**
   - Privilégiez les ratios > 7:1
   - Utilisez les opposés (noir/blanc, bleu foncé/blanc)

2. **Testez sur différents écrans**
   - iPhone avec luminosité variable
   - Mode sombre / Mode clair
   - Conditions d'éclairage différentes

3. **Pensez à l'accessibilité**
   - Daltonisme (8% des hommes)
   - Vision réduite
   - Luminosité extérieure

4. **Utilisez les préréglages**
   - Blanc (#FFFFFF) pour fonds foncés
   - Noir (#000000) pour fonds clairs
   - Gris clair (#E5E5EA) pour variation
   - Gris foncé (#3A3A3C) pour élégance

### Évitez Ces Erreurs

❌ **Texte gris sur fond gris**
❌ **Couleurs trop similaires**
❌ **Texte clair sur fond clair**
❌ **Texte foncé sur fond foncé**
❌ **Saturation extrême** (éblouissant)

### Préférez

✅ **Blanc sur foncé** ou **Noir sur clair**
✅ **Contraste fort** (ratio > 7)
✅ **Couleurs complémentaires**
✅ **Simplicité** (2 couleurs max)
✅ **Test réel** sur appareil
