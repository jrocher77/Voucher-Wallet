# 🎨 Guide Rapide : Personnalisation des Couleurs de Texte

## Qu'est-ce qui a changé ?

Vous pouvez maintenant personnaliser **à la fois** :
- ✅ La couleur de fond de la carte
- ✅ **NOUVEAU** : La couleur du texte sur la carte

## 🚀 Comment ça marche ?

### Lors de l'ajout d'un bon

1. **Scannez ou saisissez** les informations du bon
2. **Descendez** jusqu'à la section "Couleur de la carte"
3. **Choisissez** :
   - La couleur de fond avec le premier ColorPicker
   - La couleur du texte avec le second ColorPicker
4. **Vérifiez** l'aperçu en temps réel
5. **Enregistrez** !

### Lors de l'édition d'un bon

1. **Tapez** sur un bon existant
2. **Tapez** sur le bouton "Modifier"
3. **Descendez** jusqu'à "Couleur de la carte"
4. **Changez** les couleurs comme vous voulez
5. **Enregistrez** !

## ⚠️ Validation automatique

### Le système vous avertit si :
- Les couleurs sont identiques
- Les couleurs sont trop similaires (contraste insuffisant)
- Le texte risque d'être difficile à lire

**Exemple :**
```
⚠️ Les couleurs sont trop similaires. 
   Le texte sera difficile à lire.
```

### Standards de contraste :
- ✅ **Bon contraste** : Ratio ≥ 3:1 (recommandé : 4.5:1)
- ❌ **Contraste faible** : Ratio < 3:1

## 🎨 Préréglages disponibles

### Couleurs de fond
15 couleurs d'enseignes françaises populaires :
- Carrefour (bleu)
- Auchan (rouge)
- Leclerc (bleu et blanc)
- Casino (rouge/vert)
- Fnac (jaune)
- Et plus...

### Couleurs de texte
4 couleurs optimisées :
- **Blanc** (#FFFFFF) - Pour fonds foncés
- **Noir** (#000000) - Pour fonds clairs
- **Gris clair** (#E5E5EA) - Alternative douce
- **Gris foncé** (#3A3A3C) - Alternative élégante

## 🤖 Apprentissage automatique

### Le système apprend vos préférences !

**Exemple :**
1. Vous créez un bon "Carrefour" avec fond bleu et texte blanc
2. La prochaine fois que vous ajoutez un bon "Carrefour"...
3. **Magie !** Les couleurs sont automatiquement pré-remplies ! ✨

### Suggestion intelligente

Si vous importez un PDF avec une nouvelle enseigne :
- Fond **clair** détecté → Texte **noir** suggéré
- Fond **foncé** détecté → Texte **blanc** suggéré

## 📱 Aperçu en temps réel

### Ce que vous voyez :
Une mini-carte qui affiche :
- Le nom de l'enseigne
- Le numéro du bon
- Le montant (si renseigné)
- **Exactement** comme elle apparaîtra dans la liste !

### Mise à jour instantanée
Chaque changement de couleur met à jour l'aperçu **immédiatement**.

## 💡 Conseils d'utilisation

### Pour une bonne lisibilité :
1. **Fonds clairs** → Utilisez du texte **foncé**
2. **Fonds foncés** → Utilisez du texte **clair**
3. **Testez** avec l'aperçu avant d'enregistrer
4. **Écoutez** les avertissements du système

### Exemples de bonnes combinaisons :
- ✅ Fond bleu foncé (#0055A5) + Texte blanc (#FFFFFF)
- ✅ Fond jaune (#FFD700) + Texte noir (#000000)
- ✅ Fond vert foncé (#006400) + Texte blanc (#FFFFFF)
- ✅ Fond gris clair (#E5E5EA) + Texte gris foncé (#3A3A3C)

### Exemples de mauvaises combinaisons :
- ❌ Fond blanc (#FFFFFF) + Texte blanc (#FFFFFF)
- ❌ Fond noir (#000000) + Texte noir (#000000)
- ❌ Fond gris (#808080) + Texte gris (#888888)
- ❌ Fond jaune clair (#FFFFAA) + Texte blanc (#FFFFFF)

## 🔧 Accessibilité

Cette fonctionnalité respecte les normes **WCAG 2.1** :
- Niveau A : Contraste minimum 3:1 ✅
- Recommandé AA : Contraste 4.5:1 pour texte normal
- Recommandé AAA : Contraste 7:1 pour texte amélioré

## ❓ FAQ

### Q : Puis-je utiliser n'importe quelle couleur ?
**R :** Oui ! Le ColorPicker iOS vous donne accès à toute la palette de couleurs.

### Q : Que se passe-t-il avec mes bons existants ?
**R :** Ils utilisent automatiquement du texte blanc (#FFFFFF) par défaut.

### Q : Puis-je changer les couleurs plus tard ?
**R :** Oui ! Modifiez un bon à tout moment via le bouton "Modifier".

### Q : L'apprentissage fonctionne-t-il aussi pour les couleurs de texte ?
**R :** Oui ! Le système mémorise vos préférences de couleur de texte par enseigne.

### Q : Comment réinitialiser les couleurs apprises ?
**R :** Allez dans les statistiques d'apprentissage (à venir) pour réinitialiser.

### Q : Les couleurs sont-elles synchronisées entre appareils ?
**R :** Pas encore. C'est une amélioration future avec iCloud.
