## 📝 Glossaire

### Termes utilisés dans la documentation

| Terme | Définition |
|-------|------------|
| **Contraste** | Différence de luminosité entre fond et texte |
| **WCAG** | Web Content Accessibility Guidelines (standards d'accessibilité) |
| **Ratio de contraste** | Mesure numérique du contraste (ex: 4.5:1) |
| **Luminosité** | Clarté d'une couleur (0 = noir, 1 = blanc) |
| **Hex** | Format de couleur hexadécimal (#FFFFFF) |
| **Apprentissage** | Mémorisation des préférences utilisateur |
| **Suggestion** | Proposition automatique de couleur |
| **Validation** | Vérification du contraste suffisant |
| **Aperçu** | Visualisation en temps réel |
| **Préréglage** | Couleur prédéfinie |
