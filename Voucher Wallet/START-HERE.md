# 🚀 DÉMARRAGE EN 60 SECONDES

## ✅ Ce qui a été fait

Vous pouvez maintenant **choisir la couleur du texte** sur vos cartes de bons, en plus de la couleur de fond !

## 🎯 Test rapide

### 1️⃣ Lancer l'app
```
Cmd + R
```

### 2️⃣ Créer un bon
```
+ → Saisie manuelle
```

### 3️⃣ Descendre à "Couleur de la carte"
```
🎨 Couleur de fond     [Picker bleu]
🎨 Couleur du texte    [Picker blanc]  ← NOUVEAU !
👁️ Aperçu               [Mini-carte]
```

### 4️⃣ Jouer avec les couleurs
```
Essayez :
- Blanc sur blanc → ⚠️ Avertissement !
- Bleu + Blanc    → ✅ Bon contraste
- Jaune + Noir    → ✅ Bon contraste
```

### 5️⃣ Enregistrer et admirer ! ✨

## ✨ Fonctionnalités clés

- ✅ **2 ColorPickers** : fond + texte
- ✅ **Validation automatique** : alerte si mauvais contraste
- ✅ **Aperçu temps réel** : voir immédiatement le résultat
- ✅ **Apprentissage** : mémorise vos préférences par enseigne
- ✅ **Suggestion** : propose blanc ou noir selon le fond
- ✅ **Standards WCAG** : contraste minimum 3:1

## 🎨 Exemples de bonnes combinaisons

```
✅ Bleu foncé (#0055A5) + Blanc (#FFFFFF)
✅ Jaune (#FFD700) + Noir (#000000)
✅ Rouge (#E30613) + Blanc (#FFFFFF)
✅ Vert foncé (#006400) + Blanc (#FFFFFF)
```

## ❓ Questions ?

### Où modifier les couleurs ?
➡️ Dans "Nouveau Bon" ou "Modifier" → Section "Couleur de la carte"

### L'app apprend mes choix ?
➡️ Oui ! Créez "Carrefour" avec texte blanc, la prochaine fois c'est automatique ✨

### Je peux tout personnaliser ?
➡️ Oui ! N'importe quelle couleur pour fond et texte

### Et les bons existants ?
➡️ Texte blanc par défaut, modifiables à tout moment

## 🆘 Problème ?

**Avertissement affiché ?**  
C'est normal ! Les couleurs sont trop similaires. Changez-en une.

**Texte invisible ?**  
Modifiez le bon et changez la couleur de texte.

**Les couleurs ne sont pas sauvegardées ?**  
Vérifiez que vous appuyez sur "Enregistrer".
